# Shipment Guaranteed Delivery

The shipment guaranteed-delivery example shows a shipment dispatch command being stored, leased, acknowledged, retried, and imported through `IServiceCollection`.

Use `AddShipmentGuaranteedDeliveryDemo()` for the focused example registration or `AddPatternKitExamples()` for the aggregate example catalog. The example includes both `ShipmentGuaranteedDeliveryQueues.Create()` and `GeneratedShipmentGuaranteedDeliveryQueue.Create()` so teams can compare fluent and generated setup.
